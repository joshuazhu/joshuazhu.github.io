Effective C# -- 6. Understand the Attraction of Small Functions

Posted on 2017-04-25 | In .Net

  1. .Net runtime invokes the JIT compiler to translate IL generated by C# compiler into machine code.The CLR invokes the JITer on function by function basis.

  2. Functions that do not ever get called do not get JITed. You can minimise the amount of code that gets JITed by factoring code into more smaller functions.

  3. Enregistration selecting which local variables can be stored in registers rather than on the stack. Fewer local variables are good for enregistration.
